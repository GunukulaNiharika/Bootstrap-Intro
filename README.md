# Bootstrap-Intro

1.Install Node\
2.To set up a new or existing npm package
```
npm init
```
3.Install lite-server
```
npm install lite-server --save-dev
```
4.Downloading Bootstrap 4
```
npm install bootstrap@4.0.0 --save
npm install jquery@3.3.1 popper.js@1.12.9 --save
```
5.run server
```
npm start
```
